# Stiffness of thin wall beams with partial delamination

This code can provide the cross sectional stiffness of closed and open cross-section thin walled beam with partial delamination. So this code helps in reducing the 3D composite beam to 1D equivalent beam. Here Mechanics of Structural genome and Variational asymtptotic methods are combined to obtain the equivalent 1D stiffness of open and closed section beams with partial delamination.

To obtain the stiffness of any cross section, generate an input file in the format described in input_file_generation, then run Main_stiffness_calculation.m file and provide the name of the input file generated when prompted to find the stiffness of the composite cross-section with partial delamination.
